# STM講座ドキュメント

東京農工大学航空研究会で、新歓や技術伝承のために使用するSTM32マイコンの講座ドキュメントです。

アカウントベースの進捗管理を行い、進捗があった場合にディスコードに通知する機能を持っています。

[ログインページはこちら](https://nokolat.github.io/2026_STM32_Document/)

## ドキュメント

- 

- [ファイル構造](./docs/fliestruct.md): ドキュメントのファイル構造と、どこを見れば何ができるかの説明
- [ドキュメントの編集](./docs/create_document.md): 講座を追加する方法の説明
- [ログインapi](./docs/api_login.md): ログインAPIの叩き方と、送る内容・受け取る内容の説明
- [進捗管理api](./docs/api_course.md): 進捗APIの叩き方と、送る内容・受け取る内容の説明

## ドキュメントの編集について

- ドキュメントは `src/pages/pages/`にあります。

- PR名は`docs/3-2/〇〇`のように、編集するページに対応させてくれるとわかりやすいです

- PRを投げるときは@aoi-netaiをレビュアーにしてください。
    - 確認次第、フィードバックやマージを行います。
